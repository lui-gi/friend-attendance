[ SETUP INSTRUCTIONS ]

  1. Create a new Google Sheet: sheets.new
  2. Name it whatever you want (e.g., "Friend Attendance")
  3. Add these column headers in Row 1:
    Date | Day | Friends
  4. Go to Extensions → Apps Script
  5. Delete any code there and paste this:
  1. Click Deploy → New deployment
  2. Click gear icon → Select "Web app"
  3. Description: "Friend Tracker API"
  4. Execute as: Me
  5. Who has access: Anyone
  6. Click Deploy
  7. Copy the Web app URL (looks like: https://script.google.com/...)
  8. Paste it below and click Save!
TROUBLESHOOTING:
  • If you get "[ERROR] Failed to fetch" - make sure you set "Who has access" to Anyone
  • After deploying, you may need to authorize the script (Google will show a scary warning - click "Advanced" then "Go to [project name]")
  • If you update the Apps Script code, you need to create a NEW deployment (not edit the old one)
  • The URL should end with /exec NOT /dev
  • Test the URL by pasting it in your browser - it should say "OK"
